# market-data-bus

A small C++ message bus / market-data-engine project (CMake-based).

## Build

This project uses CMake. From the repository root:

```bash
mkdir -p build
cd build
cmake ..
cmake --build .
```

## Run tests / examples

After building, binaries are in the `build/` and `build/engine` subfolders.
